Messaging Module – Setup

Step 1 – Create public folder

Create a folder to serve web content separately:

mkdir -p /var/www/tutorial/containers/public

Step 2 – Copy index.php

Copy/paste the basic HTML index.php provided to /public/

<!DOCTYPE html>
<html>
<head>
    <title>Messaging Lessons</title>
</head>
<body>
    <h1>Messaging Lessons</h1>
    <ul>
        <li><a href="sender.php">Send Message</a></li>
    </ul>
</body>
</html>

This will allow your web server to display a simple landing page with links to exercises.

Step 3 – Start PHP Built-in Server

Open terminal in /var/www/tutorial/containers and run:

php -S localhost:8000 -t public/

Open browser at http://localhost:8000 to preview the index page.
