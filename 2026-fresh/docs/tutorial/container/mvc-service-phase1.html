Service Layer – Phase 1

Objective

In this phase, you will create a new class: MessageService.

  • No logic will be moved yet
  • No controller changes yet
  • We are preparing architectural structure
This phase is about structure, not behavior.

What Must Change

Create the services directory using Git Bash:

mkdir /var/www/tutorial/containers/app/services/

Then create the file:

touch /var/www/tutorial/containers/app/services/MessageService.php
Do not modify existing files yet.

AI Prompt

We are starting Service Layer Phase 1. Create MessageService.php inside: app/services/ Requirements: - Plain PHP class - No namespaces - No external dependencies - Include basic comments - Add empty method send($type, $message) - Do not move any logic yet
The AI must only generate the new file. It must not modify the controller yet.

What To Check

  • File exists in app/services/
  • Class name is MessageService
  • Method send($type, $message) exists
  • No CLI logic inside yet
  • No syntax errors
Run your server and confirm the application still works exactly the same.
