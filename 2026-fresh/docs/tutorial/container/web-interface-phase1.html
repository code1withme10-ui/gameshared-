Phase 1 – Create Sender Page

Use AI to generate your first HTML form for sending messages

Objective

In this phase, you will generate sender.php as a simple HTML page with:

  • A select box for send type (SMS, Email, WhatsApp)
  • A text area to enter the message
  • A submit button

No PHP processing yet — just static HTML. This allows you to start interacting with AI in a controlled way.

Step 1: Open Your Workspace

Make sure your workspace is the tutorial folder:

cd /var/www/tutorial/containers

All generated files must live here. This isolates your work and keeps it organized.

Step 2: Generate Basic HTML

Use the AI to generate sender.php. Provide this prompt:

"Create a basic HTML page named sender.php with a select box for send type (SMS, Email, WhatsApp), a text area for message input, and a submit button. Do not include any PHP code. Keep the page simple and professional."

Copy the AI output directly into sender.php inside your tutorial folder.

Step 3: Verify Your Page

Start the PHP built-in server to preview your page:

cd /var/www/tutorial/containers
php -S localhost:8000 -t public/

Open your browser at:

http://localhost:8000/sender.php

You should see your select box, text area, and submit button. No functionality yet — just layout.

Step 4: Reflection

This exercise teaches you how to start small and control AI output. In the next phase, you will convert this static HTML into a functional sender page using PHP.
