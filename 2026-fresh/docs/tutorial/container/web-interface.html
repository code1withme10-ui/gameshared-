CodeWithMe – Architectural Training Series

Step-by-step professional workflow for building flexible messaging systems

Step 1: Prepare Your Environment

  • Create a new folder: /var/www/tutorial/containers/
  • Do NOT use your existing work sandbox
  • Place all tutorial files inside this folder
This tutorial must be completely isolated from production or client work.

Step 2: Reset Your AI Session

Before starting the exercise:

  • Log out of your AI account
  • Log back in
  • Start a brand new chat session
  • Do NOT attach the chat to any existing project
Previous chat history may influence AI responses and break tutorial constraints.

Step 3: Initialize the AI Correctly

Download the following files and use them in a brand new AI chat session:

  • Download prompt.md
  • Download SOP_web.txt

Instructions:

  1. Open prompt.md, copy all contents, paste into AI, and send
  2. Open SOP_web.txt, copy all contents, paste into AI, and send
Professionals control their tools. SOP prevents AI from drifting into creativity or complexity.

Step 4: Follow Phases Strictly

This tutorial is divided into phases. Do not skip:

  • Phase 1 – Generate basic sender.php HTML (form only)
  • Phase 2 – Convert sender.php to POST form and execute CLI
  • Phase 3 – Introduce container if needed
  • Phase 4 – Add additional channels like WhatsApp
Architectural understanding comes from step-by-step progression.

Step 5: Preview Your Work

Start the PHP built-in server in your tutorial folder:

cd /var/www/tutorial/containers
php -S localhost:8000 -t public/

Open your browser and navigate to:

http://localhost:8000/sender.php
This allows you to preview and test your sender.php page as you progress through the phases.

Professional Conduct

  • Write code yourself
  • Read every line generated by AI
  • Understand before copying
  • Follow phased prompts exactly

You are here to **learn architecture and controlled development**, not to build a platform.
