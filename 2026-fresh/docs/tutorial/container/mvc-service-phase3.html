Service Layer – Phase 3

Objective

Move message storage logic from MessageController into MessageService.

After this phase, the Controller will only coordinate the request.

What Must Change

Your Controller currently:

  • Calls Service for CLI execution
  • Calls Model to save message

Now:

  • Service must call the Model
  • Service must return result to Controller
  • Controller must stop calling Model directly
Do not modify routing. Do not introduce dependency injection. Do not redesign project structure.

  • Check show page still works
If everything works, Service Layer is now fully functional.

Architecture Now

Browser ↓ Router ↓ Controller ↓ Service ↓ ↓ CLI Model ↓ messages.json

The Controller is now lightweight. Business logic lives in Service.
