Phase 1 – Hard-Coded Implementation

Feel the pain of tight coupling. Learn to use AI as a professional tool.

Objective

In this phase, you will manually create the EmailNotifier and NotificationService in index.php. This helps you experience why tightly coupled code is rigid and hard to extend.

Instructions

  1. Open index.php.
  2. Require EmailNotifier.php and NotificationService.php.
  3. Manually create objects: 
    $notifier = new EmailNotifier();
$service  = new NotificationService($notifier);
$service->send($message);
  4. Run the CLI command to test: 
    php index.php email "Hello World"

Terminology

Tight Coupling:When a class depends directly on another specific class, making it hard to change or extend.

Notifier:A component responsible for sending messages through a specific channel (e.g., Email).

Sample AI Prompt
