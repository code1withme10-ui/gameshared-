Validation testing
Validation Test Cases
Email field: Must accept only valid email formats.
Phone field: Must accept only numeric characters, valid length.
DOB field: Must not accept future dates.
Grade category: Must match calculated age from DOB.
Document uploads: Validate file type, size, and optional preview.
