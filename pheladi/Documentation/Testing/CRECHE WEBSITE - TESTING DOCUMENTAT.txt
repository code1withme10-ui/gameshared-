CRECHE WEBSITE - TESTING DOCUMENTATION

Introduction
The purpose of the documentation is to ensure that the websites features are functioning correcting and meet the requirements listed.
The website is supposed to do the following:
Allow a parent to login and add more than one children.
Allow the headmaster to be able to accept or reject a child.
The parent must be able to track the status of the child if whether they are accepted or rejected.

Test Environment:
Browse: Chrome, Firefox, Edge
Devices: Laptop
Database: JSON
