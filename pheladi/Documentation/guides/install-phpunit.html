1. Introduction

This guide introduces you to PHPUnit testing, a framework used to automate the testing of PHP code. It helps you avoid repetitive manual testing and ensures consistent results.

IMPORTANT: Throughout this guide, replace mpho with your own name in the folder path. Example: /var/www/gameshared/mpho/

This guide assumes:

  • You are using Windows
  • You are using Git Bash
  • Your working folder is in: /var/www/gameshared/<yourName>/
