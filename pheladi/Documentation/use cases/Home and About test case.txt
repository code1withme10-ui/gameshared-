CRECHE WEBSITE – TESTING DOCUMENTATION
Home Page & About Page
Introduction

The purpose of this document is to ensure that the Home Page and About Page of the crèche website function correctly, display accurate information, and provide a user-friendly experience for parents, guardians, and visitors.

Test Environment

Browsers: Chrome, Firefox, Edge
Devices: Laptop, Tablet, Mobile
Database: JSON / Static Content

Home Page – Functional & UI Testing
_____________________________________________________________________________________________________________________________________
Test-ID	        Test Case Description	                  Steps                                       Expected Results

TC-HP-02	Crèche name and logo display	          Open home page	                      Logo and crèche name are visible
TC-HP-03	Navigation menu visibility	          View navigation bar	                      All menu items are visible

TC-HP-08	Images load correctly	                  Observe page images	                      Images load without distortion
TC-HP-09	Footer information display	          Scroll to footer	                      Contact info and copyright visible
TC-HP-10	Responsive layout	                  Resize browser	                      Layout adapts correctly
